# 코딩테스트
> 코딩 테스트는 소프트웨어 관련 기업에서 지원자의 기술적인 역량을 시험하기 위해 실시하는 알고리즘이나 직무 연관성이 있는 기술에 대한 구현 테스트를 의미

### coding test study rules

1. 자료구조 및 알고리즘 개념 이해
2. 자료구조 및 알고리즘을 활용한 문제풀이
3. 풀어본 문제 노트정리

## Description
