# map 함수

## map() 함수란?
* 파이썬 내장 함수로 각 요소들에 특정한 함수를 적용 기능을 제공한다.

## 사용법
> 리스트 및 딕셔너리 등 자료형에 일괄적으로 함수를 적용하고 싶을때 사용된다.
### 문자열을 일괄적으로 정수로 변환
```python
item = "1 2 3 4"
list(map(int,item.split()))# [1,2,3,4] 로 변환
```
### 리스트 자료형 안에 있는 자료들을 lambda 함수로 정의한 함수를 일괄 적용
```python
item = [[1,2],[2,3],[4,5]]
list(map(lambda x:(x[0]+x[1]),item)) # [3,5,9]로 변환
```
